## About
A modern, interactive pizza ordering application built with React. This application provides users with a seamless experience to browse delivery locations via an interactive map and securely input payment details using a dynamic credit card form.

## üöÄ Features

*   **Interactive Menu:** Browse through various pizza categories with a clean, grid-based UI.
*   **Shopping Cart:** Full state management for adding, removing, and updating pizza quantities.
*   **Location Services:** Integrated map using `react-leaflet` showing branch locations across 5 major cities.
*   **Moderm Payment Experience:** Dynamic credit card form with `react-credit-cards-2` featuring real-time validation and visual card preview.
*   **Responsive Design:** Fully optimized for mobile, tablet, and desktop using Tailwind CSS.
*   **SPA Routing:** Seamless navigation with `react-router-dom` using subfolder-aware configuration.
